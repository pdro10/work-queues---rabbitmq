# Como executar o conteúdo:

primeiro clone o repositorio,
Para executar você precisará ter o python e o rabbitmq instalado e ativo

# Como usar:

Instale PIKA: 
python -m pip install pika --upgrade

Após isso:

python worker.py

# Terminal 2: 
python worker.py

python new_task.py First message.
